#twoclick social sharing

Diese Erweiterung bringt das jQuery Plug-In 'socialshareprivacy' von Heise auf den firtz.
Das Original findet Ihr unter: http://www.heise.de/extras/socialshareprivacy/

Etwaige Anpassungen müssen im twoclick.php vorgenommen werden.

Diese Erweiterung benötigt mindestens firtz 2.0
